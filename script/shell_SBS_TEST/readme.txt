*
* 程序运行示例
*	./DER.sh SpeakerSeperation.jar ./testData ./result s
*	
* 参数1  被测试的程序包
* 参数2  音频文件的第一级目录
* 参数3  结果文件保存目录
* 参数4  s/-s将程序标注的结果导出到结果文件中
*		 ---- 其它值不存在程序标注结果
* 
* 
* 备注：
* 	1. SpeakerSeperation.jar 标注的结果临时保存在测试文件夹中，使用者需拥
*	   有测试文件夹写权限
* 	2. 脚本会删除程序的结果文件，以使测试音频文件夹复原  ----参数4使结果导
*	   出到结果目录(参数2)
* 	3. 输出结果 
* 			out.txt		为perl的测试结果
* 			out.result 	为结果的处理文件
*
